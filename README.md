FreskoExpress: Plataforma de Gestión Logística para Alimentos Perecederos
📋 Descripción del Proyecto
FreskoExpress es una solución integral de software diseñada para digitalizar y optimizar la cadena de suministro de una distribuidora de alimentos (frutas, verduras, lácteos y cárnicos). El sistema reemplaza procesos manuales y hojas de cálculo por una arquitectura robusta capaz de monitorear la cadena de frío en tiempo real y gestionar la logística en tres ciudades principales.
Este proyecto se desarrolla bajo el marco de la asignatura Arquitectura de Software de la Fundación Universitaria Konrad Lorenz.
🚀 Objetivos Principales
Calidad y Estándares: Implementar metodologías y buenas prácticas para minimizar riesgos y asegurar un producto de alta calidad.
Arquitectura Robusta: Diseñar una arquitectura basada en requerimientos no funcionales, patrones y estilos arquitectónicos modernos.
Innovación Técnica: Integrar tecnologías como sensores IoT, microservicios y sincronización offline.
🛠️ Funcionalidades Clave
Gestión de Proveedores: Registro legal, validación de certificados sanitarios y asignación de estados de aprobación.
Planificación de Rutas: Optimización de entregas considerando tipo de vehículo (refrigerado/no refrigerado) y tráfico en tiempo real.
Monitoreo IoT: Seguimiento de temperatura y humedad en tiempo real con alertas automáticas para garantizar la cadena de frío.
Inventario Inteligente: Control de stock mediante lotes y fechas de vencimiento, con rotación basada en el método FIFO (First In, First Out).
Pedidos y Facturación: App/Web para clientes con seguimiento de pedidos en tiempo real y múltiples pasarelas de pago.
Mantenimiento Predictivo: Alertas preventivas para la flota vehicular basadas en telemetría y kilometraje.
🏗️ Atributos de Arquitectura (RNFs)
Escalabilidad: Arquitectura de microservicios preparada para un crecimiento del 300% en el volumen de pedidos.
Disponibilidad: Operación 24/7 con un 99.8% de disponibilidad.
Rendimiento: Soporte para más de 500 dispositivos IoT enviando datos simultáneamente.
Resiliencia: Capacidad de sincronización offline para conductores en zonas con baja conectividad móvil.
Seguridad: Autenticación multifactor y cifrado de datos sensibles.
🔌 Integraciones
API REST: Para conexión con ERPs de clientes corporativos.
Protocolos IoT: Comunicación mediante MQTT/WebSockets para sensores de temperatura.
Geolocalización: Integración con servicios de tráfico y mapas.
